SQL Injection Wargame

단계별 학습

기본 인증 우회부터 데이터베이스 정보 수집까지 5단계로 구성된 체계적인 SQL Injection 학습 과정입니다.

실습 환경

다양한 시나리오에서 SQL Injection을 연습할 수 있는 실습 환경입니다. 사용자, 상품, 주문 데이터를 대상으로 실습하세요.

Level 1

기본적인 SQL Injection 우회 기법을 연습할 수 있는 로그인 문제입니다.
admin 계정의 FLAG를 노려보세요.

Level 2

블라인드 SQL Injection(참/거짓만 반환) 환경에서
admin FLAG를 추출하는 실전 문제입니다.

Level 3

WAF(웹 방화벽) 필터링을 우회하여
admin FLAG를 얻는 고급 SQL Injection 문제입니다.

Level 4

블라인드 SQL Injection을 활용하여
admin FLAG를 추출하는 최종 문제입니다.

Level 5

입력값이 카이사르 암호(+3)로 변환되어 쿼리에 삽입되는
특수한 SQL Injection 문제입니다.
admin FLAG를 노려보세요.
